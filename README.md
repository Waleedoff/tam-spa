# Frontend Setup Guide

This document explains how to set up and run the frontend project locally.

## 🛠️ Requirements

- **Node.js** (v16 or later recommended)
- **npm** or **yarn** (depending on the project)

---

## 🚀 Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/Waleedoff/tam-spa
cd tam-spa

```

## Install Dependencies

```bash
npm install
```


## Run Tam Website

```bash
npm run
```
